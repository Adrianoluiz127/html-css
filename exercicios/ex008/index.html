Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo para deixar o texto grande atualmente podemos usar a tag "big", mas ela já se tornou obsoleta e pode cair em desuso ou parar de funcionar em breve. para deixar a letra pequena podemos usar a tag "small".

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado utilizando a tag "del".

Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adiconado depois utilizando a tag "ins".
